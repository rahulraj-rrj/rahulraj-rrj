# 🚀 About Me

Backend-focused **Computer Science Engineer** passionate about designing scalable, production-ready applications.

I enjoy building systems that remain reliable under load, prioritizing clean architecture, maintainable code, and thoughtful API design over unnecessary complexity.

My primary interest lies in **Java Backend Engineering**, where I work extensively with **Spring Boot**, **Spring Security**, **JWT**, **Docker**, **MySQL**, and **React** to develop complete end-to-end applications.

My flagship project, **Encore**, demonstrates production-oriented concepts including optimistic locking, stateless authentication, role-based authorization, cloud deployment, and distributed system fundamentals.
